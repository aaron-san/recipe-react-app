# Workflow

## Run development server

```shell
npm start
```

## Deploy

```shell
git add .
git commit -m "message"
git push
npm run build
firebase deploy
```

```shell
firebase login
```

# If there are login problems, try to reathorize

```shell
firebase login --reauth
```
